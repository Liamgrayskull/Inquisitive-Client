Welcome to Inquisitive! This is where all documentation will be sourced, updated with each update of the pack, or server update, to teach you about Inquisitive's realms.

Documentation is split into groups. Most groups are self explanitory, but Lore details server-specific details about the happenings of the world, which will update most frequently.

Any contributions you want to add to documentation is welcome! Just open a ticket about it!
